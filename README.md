# hrid's split keyboard

hrid's split keyboard is a sleek, custom staggered ortholinear wireless split keyboard that runs off the ESP32-S3-WROOM SMD module.

# Features

- OLED display
- EC11 rotary encoder w/ button
- Hot-swappable key switches
- Exposed circuitry look w/ acrylic top plate
- USB-C charging
- 2000mAh li-ion battery
- ESP32-S3-WROOM-1 MCU
